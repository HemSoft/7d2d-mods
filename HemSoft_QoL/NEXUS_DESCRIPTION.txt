# Nexus Mods Description
# Copy this BBCode to the Nexus mod description field

[center][size=5][b]HemSoft QoL[/b][/size]
[i]Quality of Life improvements for 7 Days to Die V2.5[/i][/center]

[size=4][b]Features[/b][/size]

[b]Inventory Hotkeys[/b]
Keyboard shortcuts for inventory management when a container is open:

[list]
[*][b]Q[/b] - Quick Stack: Deposit items that match existing stacks in the container
[*][b]Alt+X[/b] - Stash All: Deposit all items from your backpack into the container
[*][b]Alt+R[/b] - Restock: Pull items from container to fill your existing stacks
[*][b]S[/b] - Sort Container: Sort container items alphabetically
[*][b]Alt+M[/b] - Sort Inventory: Sort your backpack alphabetically (disabled by default)
[/list]

[i]Note: Vanilla uses [b]R[/b] for Loot All. All hotkeys are fully configurable![/i]

[b]HUD Info Panel[/b]
Displays real-time player stats on your HUD:

[list]
[*][b]Level[/b] - Your current player level
[*][b]Gamestage[/b] - Affects enemy difficulty and spawns
[*][b]Lootstage[/b] - Affects loot quality
[*][b]Day[/b] - Current in-game day
[*][b]Blood Moon[/b] - Countdown to next blood moon (color-coded!)
[*][b]Kills[/b] - Total zombie kills
[*][b]Nearest Enemy[/b] - Distance and name of closest hostile (color-coded by distance!)
[*][b]Enemy Count[/b] - Number of hostiles within 100m
[/list]

[b]Console Commands[/b]
Type [code]hs[/code] in the console (F1) to:
[list]
[*]View current settings and hotkey configuration
[*]Toggle individual info panel elements on/off
[*]Use [code]hs all[/code] or [code]hs none[/code] to show/hide all panel elements
[/list]

[b]Panel Positioning[/b]
Move the info panel anywhere on your screen:

[list]
[*][b]Alt+P[/b] - Toggle position mode
[*][b]Arrow Keys[/b] - Move panel (hold Shift for faster)
[*][b]Alt+P[/b] again - Save position and exit
[/list]
Position is saved and persists across game sessions!

[size=4][b]Installation[/b][/size]

[list=1]
[*]Download and extract the mod
[*]Copy the [b]HemSoft_QoL[/b] folder to your [code]7 Days to Die/Mods/[/code] directory
[*]Launch the game with [b]EAC DISABLED[/b] (select in Steam launcher)
[/list]

[size=4][b]Configuration[/b][/size]

[b]Hotkeys:[/b] Edit [code]Config/HemSoftQoL.xml[/code] to customize hotkeys or disable them.
[b]Info Panel:[/b] Edit [code]Config/InfoPanelConfig.xml[/code] or use the [code]hs[/code] console command.

[size=4][b]Requirements[/b][/size]

[list]
[*]7 Days to Die V2.5+
[*]EAC must be disabled (required for DLL mods)
[/list]

[size=4][b]Compatibility[/b][/size]

[list]
[*]Should be compatible with most other mods
[*]Works in single player and multiplayer (all players need the mod)
[*]Compatible with SMX UI and other UI overhauls (info panel can be repositioned)
[/list]
