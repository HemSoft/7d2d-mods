# Images for HemSoft QoL

This folder contains images for the mod page and documentation.

## Required Images for Nexus

### banner.png
- **Size**: 1280x720 or similar 16:9 ratio
- **Content**: Mod name, key features, game screenshot background
- **Used**: Main mod page header

### screenshot_panel.png  
- **Size**: 1920x1080 (or in-game resolution)
- **Content**: Screenshot showing the HUD info panel in action
- **Used**: Nexus gallery, README

### screenshot_inventory.png
- **Size**: 1920x1080 (or in-game resolution)  
- **Content**: Screenshot showing inventory with container open (highlight hotkeys)
- **Used**: Nexus gallery

## Nexus Mod Page Requirements

1. **Main Image**: At least 1280x720, shown at top of mod page
2. **Gallery Images**: Show features in action
3. **Thumbnail**: Auto-generated from main image

## Tips for Good Screenshots

1. Use high graphics settings
2. Capture during daytime for visibility
3. Show the UI elements clearly
4. Consider adding text overlays to highlight features
