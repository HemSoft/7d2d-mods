# HemSoft QoL - Quality of Life mod for 7 Days to Die V2.5

## Inventory Hotkeys (when container is open)

- **Q** - Quick Stack: deposits items matching existing container stacks
- **Alt+X** - Stash All: deposits all backpack items
- **Alt+R** - Restock: pulls items to fill partial backpack stacks
- **S** - Sort Container: sorts container items alphabetically
- **Alt+M** - Sort Inventory: sorts backpack items (disabled by default)

> Note: Vanilla uses **R** for Loot All. All hotkeys are configurable in `Config/HemSoftQoL.xml`

## HUD Info Panel

Displays real-time stats:

- Level, Gamestage, Lootstage
- Current Day, Blood Moon countdown
- Zombie Kills
- Nearest Enemy (name + distance, color-coded by danger)
- Enemy Count (hostiles within 100m)

### Panel Controls

- **Alt+P** to enter position mode
- **Arrow Keys** to move panel
- **Shift** for faster movement
- **Alt+P** again to save and exit

### Console Commands

Type `hs` in console (F1) to:

- View current settings and hotkey configuration
- Toggle elements: `hs level`, `hs enemy`, `hs bloodmoon`, etc.
- Show/hide all: `hs all` or `hs none`

## Requirements

- 7 Days to Die V2.5
- EAC disabled

## Installation

Extract to `7 Days to Die/Mods/` folder.
