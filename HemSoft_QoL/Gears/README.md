# HemSoft QoL Gears Icon

This folder should contain the mod icon for the Gears settings UI.

## Required File
- `HemSoftQoL_Icon.png` - 64x64 or 128x128 PNG icon

## Icon Guidelines
- Square format (64x64 or 128x128 pixels)
- PNG format with transparency
- Should represent the mod's functionality (inventory/QoL theme)
- Will be displayed in the Gears Mod Settings menu

## Temporary Workaround
If no icon is present, Gears will use a default icon.
You can remove the `<Icon>` line from ModInfo.xml to avoid warnings.
