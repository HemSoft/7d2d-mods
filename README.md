# HemSoft 7D2D Mods

Quality of Life mods for **7 Days to Die** by HemSoft Developments.

## Mods

| Mod | Description |
|-----|-------------|
| [HemSoft_QoL](./HemSoft_QoL/) | Inventory hotkeys and QoL improvements |

## Requirements

- 7 Days to Die V2.5+
- **EAC must be disabled** for DLL mods

## Installation

Copy the mod folder(s) to your `7 Days to Die/Mods/` directory.
